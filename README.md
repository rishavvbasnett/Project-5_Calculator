# Calculator

A simple web-based calculator built as part of The Odin Project Foundations course.

## Features

- Addition, subtraction, multiplication, division
- Exponentiation (`^`) and percentage (`%`)
- Chained calculations (e.g. `5 + 3 - 1 =`)
- Negative number input (e.g. `-3 + 5`)
- Decimal support
- Delete last character (Del)
- Clear all (AC)

## How to Use

Open `index.html` in a browser and start calculating.

## Built With

- HTML
- CSS
- JavaScript (vanilla)
